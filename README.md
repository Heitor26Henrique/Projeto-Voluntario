# 🤝 Conecta Voluntário

## 📖 Sobre o Projeto

O **Conecta Voluntário** é uma plataforma web desenvolvida para aproximar instituições sociais de pessoas interessadas em realizar trabalhos voluntários.

A aplicação permite que instituições cadastrem suas necessidades e que voluntários visualizem, pesquisem e filtrem oportunidades de acordo com o tipo de ajuda desejado.

Este projeto foi desenvolvido utilizando apenas **HTML, CSS e JavaScript**, conforme os requisitos da disciplina.

---

## 🚀 Funcionalidades

- ✅ Página Inicial (Home)
- ✅ Cadastro de Necessidades
- ✅ Validação dos campos obrigatórios
- ✅ Integração com a API ViaCEP
- ✅ Preenchimento automático do endereço
- ✅ Armazenamento utilizando LocalStorage
- ✅ Visualização dinâmica das necessidades
- ✅ Pesquisa por título e descrição
- ✅ Filtro por tipo de ajuda
- ✅ Layout responsivo
